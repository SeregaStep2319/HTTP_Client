# HTTP_Client
Простая реализация HTTP-клиента для сервера смс-сообщений.
Для запуска:
1. Склонируйте репозиторий: https://github.com/SeregaStep2319/HTTP_Client
2. Внесите корректировки в файл - config.toml: укажите адрес сервера; имя пользователя и пароль для авторизации.
3. Запустите модуль через терминал: python app.py номер_телефона_отправителя номер_телефона_получателя передоваемое_текстовое_сообщение
4. Модуль должен: считать данные из конфигурационного файла, сформировать и отправить HTTP-запрос, получить и вывести в терминал HTTP-ответ, сохранить введенные аргументы командной строки и ответ сервера в log.txt.
